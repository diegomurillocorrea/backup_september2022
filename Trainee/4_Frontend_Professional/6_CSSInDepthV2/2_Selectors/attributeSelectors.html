Attribute Selectors

E[ attr ]

This is a selector for an element that has a specific attribute with any value


E[ attr="val" ]

This is a selector for an element that has a specific attribute and value


E[ attr |= val ]

This is a selector for an element that has a value or it begins with "val-"

p[ lang |= "en" ], p[ lang |= "en-uk" ]


E[ attr ^= val ]

This is a selector for an element that has a value that starts with an specific value

a[ href ^= http ], a[ href ^= mailto ]


E[ attr ~= val ]

This is a selector for an element that has a value that is a full word

a[ title ~= "unicorn eat rainbows" THIS IS NOT A FULL WORD ], a[ title ~= "email" THIS IS A FULL WORD ]


E[ attr $= val ]

This is a selector for an element that has a value that ends with an specific value

a[ href $= pdf ]


E[ attr *= val ]

This is a selector for an element that is anywhere in the content


E[ attr = "val" i ]

This is a selector for an element that is case insensitive

input[ type="checkbox" i ], E[ foo = "val" i ]
